# ملف Excel نموذجي للاستيراد

## تنسيق الملف المطلوب

يجب أن يحتوي ملف Excel على 5 أعمدة بالترتيب التالي:

| A (التاريخ) | B (جهة مقبوضات) | C (جهة مدفوعات) | D (المبلغ) | E (طريقة الدفع) |
|-------------|-----------------|-----------------|------------|----------------|
| 01/01/2024 | راتب شركة ABC |  | 5000 | income |
| 02/01/2024 |  | كارفور سوبر ماركت | 250.50 | card |
| 03/01/2024 |  | محطة بنزين | 100 | cash |
| 04/01/2024 |  | صيدلية النهدي | 75.25 | card |
| 05/01/2024 | مكافأة عمل |  | 1000 | InCard |
| 06/01/2024 |  | مطعم الطازج | 45 | cash |
| 07/01/2024 |  | فاتورة كهرباء | 200 | card |
| 08/01/2024 |  | جامعة الكويت رسوم | 500 | card |
| 09/01/2024 |  | محل ملابس | 150 | card |
| 10/01/2024 |  | سينما | 30 | cash |

## ملاحظات مهمة:

1. **الصف الأول (Header)**: يجب أن يحتوي على أسماء الأعمدة أو يمكن تركه فارغ
2. **التاريخ**: يمكن استخدام تنسيق dd/mm/yyyy أو yyyy-mm-dd
3. **جهة مقبوضات**: املأ هذا العمود فقط للمقبوضات، واترك العمود الآخر فارغ
4. **جهة مدفوعات**: املأ هذا العمود فقط للمدفوعات، واترك العمود الآخر فارغ
5. **المبلغ**: رقم بدون رموز عملة (يمكن استخدام فاصلة للآلاف)
6. **طريقة الدفع**: استخدم القيم المحددة: card, cash, income, InCard

## كيفية إنشاء الملف:

1. افتح Excel أو Google Sheets
2. أنشئ الأعمدة الخمسة بالترتيب المطلوب
3. املأ البيانات حسب المثال أعلاه
4. احفظ الملف بصيغة .xlsx أو .xls
5. استخدم وظيفة "استيراد من Excel" في التطبيق

## أمثلة على التحليل الذكي:

- **"كارفور خضار وفواكه"** → سيتم تصنيفها تحت "سوبر ماركت"
- **"تاكسي أوبر"** → سيتم تصنيفها تحت "مواصلات"
- **"صيدلية النهدي دواء"** → سيتم تصنيفها تحت "طبي"
- **"فاتورة كهرباء الكويت"** → سيتم تصنيفها تحت "فواتير"
- **"معاملة ورقية حكومية"** → سيتم تصنيفها تحت "معاملات حكومية"
- **"أي شيء غير مفهوم"** → سيتم تصنيفها تحت "آخر" مع إضافة النص الأصلي في الملاحظات
